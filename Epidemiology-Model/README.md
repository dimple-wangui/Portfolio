# Epidemiological Modeling of Disease Transmission
**Tools:** R, deSolve

### Objective
To simulate the spread of an infectious disease within a population using the 
Susceptible-Infected-Recovered (SIR) compartmental model.

### Key Features
* **Dynamic Simulation:** Modeled how transmission (beta) and recovery (gamma) rates affect the peak of an outbreak.
* **Research Application:** Interpreted statistical outputs to predict the timing of infection peaks, a critical skill for public health surveillance and resource planning.
